# Password Game Helper

Мобильный помощник для позиционной игры с паролем. Приложение анализирует
предыдущие попытки, исключает невозможные цифры и отмечает позиции, которые
можно определить однозначно.

## Возможности
- Пароли длиной от 4 до 10 цифр;
- Автоматическая отправка комбинации после ввода последней цифры;
- Экранная и физическая цифровые клавиатуры;
- Строгая дедукция для паролей длиной до 6 цифр;
- Подсветка однозначно определённых позиций;
- Блокировка невозможных цифр для каждой позиции;
- Цветовая индикация результатов в истории;
- Удаление попыток с подтверждением и автоматическим пересчётом;
- Сохранение текущей игры в Local Storage;
- Адаптивный мобильный интерфейс;
- Установка на домашний экран как PWA;
- Работа без интернета после первого успешного запуска.

## Как работает дедукция
Для каждой попытки указывается количество цифр, которые одновременно:

1. Совпадают с цифрами загаданного пароля;
2. Находятся на правильных позициях.

Для паролей длиной до 6 цифр приложение перебирает все допустимые комбинации.
Цифра считается подтверждённой только тогда, когда она находится на одной и
той же позиции во всех оставшихся вариантах.

## Требования
- Node.js 18 или новее;
- npm.

## Установка
```bash
npm install
```

## Запуск для разработки
```bash
npm run dev
```

Чтобы открыть приложение с другого устройства в локальной сети:
```bash
npm run dev -- --host 0.0.0.0 --port 8080
```

## PWA и офлайн-режим
`vite-plugin-pwa` генерирует Web App Manifest и Service Worker во время
production-сборки. При первом открытии Service Worker кэширует HTML,
JavaScript, CSS, manifest и иконки. После успешной первичной загрузки приложение
можно открыть без подключения к интернету.

Для работы Service Worker приложение должно быть доступно через HTTPS или `localhost`.
